
# Word and Character Count

Esta extensão exibe no rodapé do VS Code a contagem de palavras e caracteres do texto inteiro ou da seleção atual.

## Como usar

1. Instale a extensão (via VSIX ou Marketplace).
2. Abra qualquer arquivo de texto.
3. Veja a contagem de palavras e caracteres no canto inferior direito do rodapé.
4. Se selecionar um trecho, a contagem será apenas da seleção.

## Funcionalidades

- Contagem automática de palavras e caracteres.
- Atualização dinâmica ao editar ou selecionar texto.
- Suporte a vários tipos de arquivo.
